Project Overview

This project presents an interactive Amazon Sales Dashboard developed in Microsoft Power BI. It helps analyze business performance using various KPIs and visualizations, allowing users to explore sales trends, profitability, customer segments, product categories, and regional performance.

Key Features
 KPI Cards (Total Sales, Profit, Orders, Units Sold)
 Year-wise Sales Trend Analysis
 Category & Sub-Category Performance
 Country-wise Sales Map
 Regional Sales Comparison
 Customer Segment Analysis
 Interactive Slicers (Category, Region, Segment, Year)
Tools Used
Microsoft Power BI
Power Query
DAX
Data Modeling
Business Insights
Monitor overall business performance through KPIs.
Identify top-performing product categories and sub-categories.
Compare sales and profit trends over multiple years.
Analyze regional and country-wise sales distribution.
Filter data dynamically using interactive slicers for deeper analysis.
